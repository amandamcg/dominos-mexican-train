In-Game Controls




Warning, undoing the last move can have bad consequences if a player joins mid game...

IE Player A makes a move, then Player B joins. If you try to roll back the move, you'll first undo Player B joining (which means Player B will then get a new hand).

Also, a player changing their train's public / private state could be undone as part of rolling back changes.
